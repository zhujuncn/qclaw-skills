# Maps Travel List

OpenClaw skill for creating Google Maps travel place lists with formatted notes.

## Features

- Create country-code named lists in Google Maps (SE, FI, EE, NO, LT, etc.)
- Search and add main attractions
- Add formatted notes with visit duration and costs
- Route-optimized numbering

## Usage

1. Install via SkillHub or clone to `~/.qclaw/skills/maps-travel-list/`
2. Trigger with keywords: "创建地点列表", "旅行规划", "Google Maps 列表"

## Note Format

```
#序号 英文名 | 时长 | 费用
```

Example:
- `#1 Vasa Museum | 2h | SEK190`
- `#2 Gamla Stan | 1h | free`
