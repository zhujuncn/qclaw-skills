# Suggestions (AI) 模块 — 自动预测 + 价值检测

## 工作逻辑

1. 基于当前赛季统计计算各市场概率
2. 与 bookmaker 赔率对比
3. 统计概率比赔率隐含概率高 >=10% → 标记为"有价值"

## 预测市场

- 1X2（含置信度%）
- O/U 0.5 / 1.5 / 2.5 / 3.5 / 4.5
- BTTS

## 信心星级

| 星级 | 条件 |
|------|------|
| 5星 | 调整后分数 >= 75% |
| 4星 | >= 65% |
| 3星 | >= 55% |
| 2星 | >= 45% |
| 1星 | < 45% |

分数 = (1X2 + O/U 2.5 + BTTS 三项置信度平均) × 样本量调整

## 颜色编码

| 颜色 | 条件 |
|------|------|
| 绿 | 置信度 >= 70% |
| 黄 | 置信度 55-70% |
| 红 | 置信度 < 55% |
| 灰 | 无数据 或 无Value（即使高置信度也灰） |

## 关键过滤器

| 过滤器 | 作用 |
|--------|------|
| Strong predictions only (>=70%) | 只看绿格 |
| Highlight only if there is a value | 只看有Value的（最严格！） |
| Min. games for prediction | 3/5(默认)/8/10 |
| Hide matches with insufficient data | 隐藏样本不足 |

## 价值检测公式

```
统计隐含概率 = 1 / 统计概率对应公平赔率
赔率隐含概率 = 1 / bookmaker赔率
Value = 统计隐含概率 - 赔率隐含概率 >= 10% → 有价值
```

## 实用建议

- 只关注**绿格+有Value**的组合
- Hover 显示详细数据（胜率、赔率等）
- 仅作研究起点，非独立策略
- 需与H2H/Statistics/Upcoming交叉验证
