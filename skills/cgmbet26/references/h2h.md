# H2H 模块 — 两队历史交锋深度对比

## 功能

- 主/客队独立筛选（主场/客场/H2H单独）
- 时间段筛选（当前赛季/近2赛季/近5场/近30天等）
- 进球时段分布图（5/10/15/30/45分钟间隔）
- 统计摘要：胜率条形图、场均进球、Clean Sheet率、先开球率
- 比分时间线条形图

## 关键用法

- 进球时段分布 → 识别"末段发力"队 vs "开局强势"队
  - 某队70-90分钟进球占60%+ → 适合末段进球投注
- 先开球统计 → 判断心理优势
- Highlight 模式：Match result / O/U 0.5-4.5 / GG/NG

## 同步模式

- 同步：改一侧自动更新另一侧
- 不同步：各面板独立过滤（如主队主场 vs 客队客场）

## 统计摘要

| 区域 | 指标 |
|------|------|
| Outcome | 胜/平/负百分比条形图 |
| Average Goals | 场均进球(绿) vs 场均失球(红) |
| Performance | Clean Sheet / 未进球 / 0-0平局 |
| First Goal Stats | 先开球率 / 先失球率 / 无球率 |
