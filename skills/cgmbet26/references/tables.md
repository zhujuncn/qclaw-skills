# Tables 模块 — 联赛积分榜与ELO走势

## 功能

- 联赛积分榜（主场/客场/全局切换）
- 球队近期战绩（进球、射门、角球、控球率）
- **ELO 走势图表**：可叠加多队对比，判断状态趋势
- 联赛级统计排名（射门、角球、GG率、O/U率）
- 分钟范围过滤（1-45 半场、46-90 下半场等）

## 关键用法

- ELO走势上行 + 对手ELO下行 → 支撑 LAY 对手策略
- 联赛GG率 > 55% → 适合 BTTS 策略
- 主场积分榜 vs 客场积分榜 → 识别"主场龙/客场虫"

## 筛选

| 过滤器 | 选项 |
|--------|------|
| 场地 | 全局 / 主场 / 客场 |
| 赛季 | 当前 / 近2赛季 / 全部 |
| 分钟 | 1-90 / 1-45 / 46-90 / 自定义 |
